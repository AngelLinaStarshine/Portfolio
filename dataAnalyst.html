"Bitcoin Price Prediction"

This project is a web application built with Flask that uses machine learning to predict Bitcoin prices. It pulls real-time price data from the CoinGecko API and uses linear regression to forecast prices for the next 30 days. The app lets calculating key metrics profit, RMSE, R-squared, and MAPE to evaluate how well the predictions perform. The web interface displays both actual and predicted prices, and users can download a report of the backtest results after verifying their identity with a SHA-256 key. To keep the predictions up to date, the app automatically fetches new data every hour.

"Yelp Polarity Reviews"

Embedding Project

This project implements a text classification model using TensorFlow and Keras to classify Yelp review data as positive or negative. It uses the Yelp Polarity Reviews dataset from TensorFlow Datasets and applies TextVectorization to preprocess the text. The model consists of an embedding layer, followed by global average pooling, and a dense layer for binary classification. After training, the embeddings are saved for analysis, and training progress is visualized with accuracy plots.
