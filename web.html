This website, styled with Bootstrap and custom fonts from Google Fonts, is clean, organized, and visually engaging, making it an effective portfolio for a professional photographer.

"Math Quiz"

This project is an engaging, interactive timed quiz application designed to challenge users with a series of multiple-choice questions. When the time expires, the quiz automatically ends, and a notification appears to inform the user. For each correct answer, the user's score increases, and once the quiz is completed, they can see how many questions they answered correctly. Visual feedback enhances the experience, with the answers being highlighted as they are selected. The application is built using a simple structure: HTML for the layout, JavaScript to manage the quiz logic and functionality, and CSS for an interactive and visually appealing design. Code review

"Quotes"

The project combines user interface design, responsive layout techniques flexbox, and interactive behavior with CSS hover effects and potential dynamic content through JavaScript. The focus is on creating a visually appealing, user-friendly page with engaging elements like quotes and navigation

"Interactive Item Highlighting on Hover"

This project demonstrates how to create an interactive UI element where individual items are highlighted when the user hovers over them. Using JavaScript and the DOM (Document Object Model), the code ensures that when the user hovers over an item, it visually stands out with a new style, while removing the highlight from any previously selected items.

"Interactive Christmas Countdown and Movie Discovery Experience"

This project is a holiday-themed interactive web application that combines a festive countdown timer, audio effects, particle animations, and a Christmas movie search and discovery tool. It creates an immersive experience that allows users to countdown to Christmas, listen to holiday music, and search for popular Christmas movies to watch. This project uses HTML, CSS, and JavaScript, along with external libraries, to create an engaging, themed web experience for Christmas.

"Guess the Number: An Interactive Challenge"

This project is an engaging number guessing game. Players are challenged to guess a random number between 1 and 50. To add a fun visual touch, the game also features interactive particle effects using tsParticles, making the experience both enjoyable and visually captivating

"Day's Advice"

This web application fetches random advice from an external API and displays it to users. The project features an interactive UI where users can click a button to receive advice, which is then read aloud using the Web Speech API for text-to-speech functionality.
