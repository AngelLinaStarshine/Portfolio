Math Quiz

This project is an engaging, interactive timed quiz application designed to challenge users with a series of multiple-choice questions. When the time expires, the quiz automatically ends, and a notification appears to inform the user. For each correct answer, the user's score increases, and once the quiz is completed, they can see how many questions they answered correctly. Visual feedback enhances the experience, with the answers being highlighted as they are selected. The application is built using a simple structure: HTML for the layout, JavaScript to manage the quiz logic and functionality, and CSS for an interactive and visually appealing design.
